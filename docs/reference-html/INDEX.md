# Tahti reference HTML — canonical structural spec

These files are the **structural ground truth** for the 16 mockup views. PNGs show what they look like; these HTMLs show what they *are*. When implementing a view in React, structure-match this HTML — same elements, same nesting, same classes (or their Tailwind equivalents per the design-system-enforcement brief).

Open any file in a browser to see it render. The CSS at the top defines every v8 token used.
